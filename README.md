# Overview
It's a website!

## Development
This website is built with [Eleventy](https://11ty.dev).

To work on the site:
* Clone this repo.
* Ensure [Node.js and Eleventy](https://www.11ty.dev/#quick-start) are installed.
* Run `npx @11ty/eleventy --serve` to view local changes live.
* In general, modify files in `src` to make changes, which will then appear in a locally in a `_site` directory. This setup is defined in `.eleventy.js`.
* Have fun!
